## About The Project

* MBTI를 알기 위해서는 딱딱한 질문식의 설문을 일일히 체크해야 합니다.
* 저희는 그런 방식보다는 챗봇과의 대화를 통해 MBTI를 알아내는 방식을 챗봇에 적용하고자 합니다.
* 저희 챗봇은 LINE Messaging API를 기반으로 만들어졌습니다.
